📊 Sales Data Analysis Dashboard

An interactive **Sales Data Analysis Dashboard** built using **Power BI** as part of the **Skill Nexus Virtual Internship**. This project combines **Python**, **SQL**, and **Power BI** to clean, analyze, and visualize sales data, providing actionable business insights.

🚀 Project Workflow
- Data Cleaning & Preprocessing using **Python (Pandas)**
- Business Analysis using **SQL**
- Interactive Dashboard Development in **Power BI**
- Business Insights & Recommendations

📈 Dashboard Features
- Executive KPIs (Sales, Profit, Gross Sales, COGS, Units Sold)
- Year-wise Sales Trend
- Product Performance Analysis
- Customer Segment Analysis
- Country-wise Profit Visualization
- Discount Band Analysis
- Interactive Filters & Slicers
- Business Insights & Recommendations

🛠️ Tech Stack
- Power BI
- Python (Pandas)
- SQL
- Power Query
- DAX

📌 Key Insights
- Sales increased by **249%** from 2013 to 2014.
- Government segment generated the highest revenue.
- Paseo and VTT contributed **45.1%** of total sales.
- Enterprise segment requires pricing and discount optimization to improve profitability.
